# Code of Conduct

## Our pledge

We want this project to be a welcoming, harassment-free space for everyone,
regardless of experience level, background, or identity.

## Expected behavior

- Be respectful and constructive.
- Assume good intent; critique ideas, not people.
- Keep discussion focused and on-topic.

## Unacceptable behavior

- Harassment, insults, or discriminatory language.
- Personal or political attacks.
- Publishing others' private information.

## Enforcement

Maintainers may remove comments, commits, issues, or PRs that violate this code,
and may block repeat offenders. To report an issue, open a GitHub issue (or, for
sensitive matters, contact the maintainer directly via their GitHub profile).

This is a small project; we handle conduct issues with common sense and a bias
toward keeping things friendly and productive.

Adapted in spirit from the [Contributor Covenant](https://www.contributor-covenant.org/).
